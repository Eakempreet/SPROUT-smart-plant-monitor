SPROUT is an IoT-based system designed to monitor plant health and provide basic protection using sensors and cloud connectivity.

## What it does
- Monitors soil moisture, temperature, and humidity
- Calculates absolute humidity for better environmental insight
- Detects motion using an IR sensor
- Triggers alerts using LED, buzzer, and servo motor
- Sends real-time data to a ThingSpeak dashboard

## Hardware Used
- NodeMCU ESP8266
- Soil Moisture Sensor
- DHT11 Temperature & Humidity Sensor
- IR Sensor
- Servo Motor
- LED & Buzzer

## Software & Tools
- Arduino IDE
- Embedded C / Arduino
- ThingSpeak IoT Cloud

## Security Note
WiFi credentials and API keys are not included in this repository.
Replace placeholder values in the code with your own credentials before uploading.
